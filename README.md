# Climate Data Analysis

## Project Overview
This project analyzes greenhouse gas emissions from different sectors (Electricity, Heating, and Transport) using Python and Pandas. The objective is to explore emission trends, clean the dataset, and generate meaningful visualizations to support data-driven decision-making.

---

## Objectives

- Clean and prepare climate emission data
- Explore sector-wise emissions
- Identify trends and patterns
- Create informative visualizations
- Practice data analysis using Python

---

## Dataset

The dataset contains emission values for different sectors and was analyzed using Microsoft Excel and Python.

---

## Tools & Technologies

- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- Excel

---

## Data Analysis Process

1. Import the dataset
2. Clean and validate the data
3. Explore the dataset
4. Calculate descriptive statistics
5. Visualize emission trends
6. Interpret the results

---

## Results

The analysis explores greenhouse gas emissions across different sectors and demonstrates how Python and Pandas  can be  used for data cleaning, exploratory data analysis, and visualization.
### Key Findings
- Transport produced the highest emissions.
- Electricity emissions remained relatively stable.
- Python and Pandas enabled efficient data cleaning and analysis.
  ### Visualizations

### Electricity Emissions
The electricity emissions showed the strongest growth  over the observed years.

### Heat Emissions
Heat-related emissions gradually increased, especially after 2005.

### Transportation Emissions
Transportation emissions rose continuously throughout the dataset.

### Combined Sector Comparison
The stacked area chart compares the contribution of all sectors over time and shows that electricity contributed the largest share.
